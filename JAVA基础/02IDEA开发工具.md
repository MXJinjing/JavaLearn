## 开发Java程序

1. 创建工程（project）
2. 创建模块（module）
3. 创建包（package）
4. 创建类（class）
5. 编写代码并启动

编译后自动放在工程的out目录

## IDEA设置

File——Settings

设置背景颜色、字体

## IDEA快捷键

- main/psvmm/sout：快速插入
- Ctrl+D：复制到下一行
- Ctrl+Y：删除所在行
- Ctrl+X：剪切所在行
- Ctrl+Alt+L：可视化代码
- ALt+Shift+上下：移动当前代码
- Ctrl+/ Ctrl+Shift+/：注释（取消）
